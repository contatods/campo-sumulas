SÚMULAS DIGITAL SCORE — PDFs POR BATERIA
=========================================

O que isso faz
--------------
Converte o ZIP de súmulas (baixado do app de súmulas, online ou local) em
PDFs prontos pra impressão, organizados assim:

  Dia/Categoria/Workout/Bateria_03.pdf   → um PDF por bateria
  Dia/00_DIA_COMPLETO.pdf                → o dia inteiro em ordem cronológica
                                           (horário → bateria → raia)
  Dia/00_FINAIS.pdf                      → só as súmulas das baterias de
                                           FINAL (as marcadas "(Final Heat)"
                                           no cronograma; requer o Excel no
                                           passo 2 — sai também se as finais
                                           ainda estiverem aguardando
                                           balizamento)

A conversão usa o navegador do seu computador (Chrome ou Edge) — o PDF sai
idêntico ao que você vê no Ctrl+P, sem deformar logo nem fontes.

O que você precisa ter instalado
--------------------------------
1. Python 3            → https://python.org  (no Windows, marque a opção
                         "Add Python to PATH" durante a instalação)
2. Google Chrome       → no Windows o Microsoft Edge (que já vem com o
                         sistema) também serve; não precisa instalar nada.

Como usar
---------
1. No app de súmulas (https://campo-sumulas.onrender.com), gere e baixe o
   ZIP do dia/evento como sempre.
2. Dê dois cliques em:
   • Mac:      "Gerar PDFs.command"
   • Windows:  "Gerar PDFs.bat"
   Abre uma página no navegador (interface local, nada vai pra internet).
3. Na página:
   • Passo 1 — escolha o ZIP na lista (mostra os recentes dos seus
     Downloads) ou arraste o arquivo pra caixa.
   • Passo 2 (opcional) — escolha o cronograma com os horários das
     baterias: o Excel de programação do evento OU o backup JSON do app
     (botão "Exportar JSON" no site). Sem ele os PDFs saem do mesmo
     jeito, só que o 00_DIA_COMPLETO.pdf fica agrupado por categoria em
     vez de seguir o horário.
   • Clique em "Gerar PDFs" e acompanhe a barra de progresso.
4. No final, clique em "Abrir pasta" — os PDFs ficam numa pasta
   "<nome-do-zip>_PDFs" ao lado do ZIP original.

Primeira execução
-----------------
• Mac: se aparecer "desenvolvedor não identificado", clique com o botão
  direito no arquivo → Abrir → Abrir (só na primeira vez). O macOS também
  pode pedir permissão de acesso a Downloads — permita.
• Windows: se o SmartScreen avisar, clique em "Mais informações" →
  "Executar assim mesmo" (só na primeira vez).

Quais arquivos preciso receber/repassar
---------------------------------------
Esta pasta (ou pelo menos estes arquivos, juntos na mesma pasta):
  gerar_pdfs.py              ← o conversor em si
  pdf_gui.py                 ← a interface local
  Gerar PDFs.command         ← atalho do Mac
  Gerar PDFs.bat             ← atalho do Windows
  ds_logo.png                ← logo (opcional, só estética)
  fonts/ (pasta)             ← fontes oficiais da interface (opcional;
                               sem elas o visual usa fontes do sistema)
  LEIA-ME — PDFs por bateria.txt

Observação: a opção de cronograma em EXCEL só funciona se a pasta tiver
também os arquivos do app completo (parsers.py etc.). Se você recebeu só o
kit acima, use o backup JSON do app como cronograma — funciona igual.
